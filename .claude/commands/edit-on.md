# /수정필요

루트에 있는 제안서 HTML 파일에서 EDIT_MODE를 true로 전환한다.

**작업 순서:**

1. 루트의 `*_proposal.html` 파일을 찾는다 (복수일 경우 가장 최근 수정 파일 선택, 또는 사용자에게 파일명 확인)
2. `</head>` 바로 전에 있는 아래 줄을 Edit한다:
   ```
   const EDIT_MODE = false;  →  const EDIT_MODE = true;
   ```
3. 완료 후 아래 메시지를 출력한다:

```
편집 모드로 전환됐습니다.

브라우저에서 Cmd+R (Windows: F5) 로 새로고침하면
상단에 편집 바가 나타납니다.

편집 완료 후 "수정 완료"를 입력하거나,
브라우저 상단 바의 "HTML 저장" 버튼을 클릭하세요.
```
