# /수정완료

루트에 있는 제안서 HTML 파일에서 EDIT_MODE를 false로 전환한다.

**작업 순서:**

1. 루트의 `*_proposal.html` 파일을 찾는다 (복수일 경우 가장 최근 수정 파일 선택, 또는 사용자에게 파일명 확인)
2. `</head>` 바로 전에 있는 아래 줄을 Edit한다:
   ```
   const EDIT_MODE = true;  →  const EDIT_MODE = false;
   ```
3. 완료 후 아래 메시지를 출력한다:

```
읽기 전용으로 전환됐습니다.

브라우저에서 Cmd+R 로 새로고침하면
편집 내용이 반영된 최종 제안서를 확인할 수 있습니다.

추가 수정이 필요하면 /수정필요 또는 "수정 필요"를 입력해 주세요.
```
